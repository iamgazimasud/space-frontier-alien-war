# Space Frontier: Alien War 🚀

A premium browser space shooter — fight through a **10-planet campaign**, take down boss battles, buy upgrades, and test yourself in endless survival mode.

## Features

- 🪐 **10-planet campaign** with escalating difficulty and boss fights
- 🔫 **Upgrade system** — weapons, shields, and ship improvements
- ♾️ **Endless survival mode** for high-score chasing
- 🎮 **Keyboard, gamepad & touch** controls — plays on desktop and mobile
- 📱 **PWA** — installable, works offline (service worker + web manifest)
- 🎨 All rendering in vanilla JS + Canvas, no frameworks

## Play Locally

```bash
# any static server works
npx serve .
```

Then open http://localhost:3000.

## Tech

Vanilla JavaScript, HTML5 Canvas, Web Audio API, Service Worker (offline support).
